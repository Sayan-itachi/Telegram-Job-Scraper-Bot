### 2ï¸âƒ£ **Environment Variables**

Create a `.env` file in the project root:

```env
# Telegram Configuration
BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
CHANNEL_USERNAME=@YourChannelUsername

# Scraping Configuration
TARGET_CHANNELS=https://t.me/s/fresheroffcampus,https://t.me/s/offcampus_phodenge,https://t.me/s/jobsinternshipswale
POST_LIMIT=10
SCROLL_DEPTH=8
DELAY_SECONDS=2

# Chrome Options
HEADLESS=true
WINDOW_SIZE=1920,1080
USER_AGENT=Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36
```

<details>
<summary><b>ğŸ“ Configuration Details</b></summary>

| Variable | Description | Default |
|:--------:|:-----------:|:-------:|
| `BOT_TOKEN` | Your Telegram bot API token | Required |
| `CHANNEL_USERNAME` | Target channel (with @) | Required |
| `TARGET_CHANNELS` | Comma-separated channel URLs | Required |
| `POST_LIMIT` | Max posts per channel | 10 |
| `SCROLL_DEPTH` | How many times to scroll | 8 |
| `DELAY_SECONDS` | Delay between posts | 2 |
| `HEADLESS` | Run Chrome in headless mode | true |

</details>

<details>
<summary><b>ğŸ”´ Common Issues</b></summary>

### âŒ **Chrome Driver Issues**
```bash
# Update Chrome
sudo apt-get update && sudo apt-get upgrade google-chrome-stable

# Clear driver cache
rm -rf ~/.wdm/
```

### âŒ **Telegram API Errors**
- Verify bot token is correct
- Ensure bot is admin in target channel
- Check rate limits (max 30 msgs/sec)

### âŒ **Parsing Failures**
- Enable debug mode to see raw messages
- Check if channel format has changed
- Update parsing rules accordingly

### âŒ **Memory Issues**
- Reduce `SCROLL_DEPTH`
- Lower `POST_LIMIT`
- Increase system swap space

</details>
